DNS服务

DNS:Domain Name Service 应用层协议，属于 C/S 结构，端口号：udp/53,tcp/53

安装包 yum install bind

一级域名
二级域名

DNS解析

Client –>hosts文件 –>DNS Service Local Cache –> DNS Server (recursion) –> Server Cache –> iteration(迭代) –> 根–> 顶级域名DNS–>二级域名DNS…
